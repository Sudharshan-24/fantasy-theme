# Fantasy for [Visual Studio Code](https://code.visualstudio.com/)

Awesome fantasy themes!

# Installation

1. Open **Extensions** sidebar panel in VS Code. "View â†’ Extensions"
2. Search for "Fantasy"
3. Click **Install**
4. Code > Preferences > Color Theme > **Fantasy**
